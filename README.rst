madrasa_branding plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/madrasafree/tutor-contrib-madrasa-branding

Usage
*****

.. code-block:: bash

    tutor plugins enable madrasa_branding
